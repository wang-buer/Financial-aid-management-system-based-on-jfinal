# 基于JFinal的助学金管理系统

## 1、项目介绍

基于JFinal的助学金管理系统拥有管理员和学生两种角色

管理员：学生管理、管理员管理、申请助学金信息管理、助学金方法等

学生：登录注册、修改个人信息、申请助学金、查看助学金发放信息等


## 2、项目技术

后端框架：JFinal

前端技术：freemarker、html、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目：是
- 硬件环境：Windows 或者 Mac OS
